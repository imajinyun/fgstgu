# 正则表达式 - Regular Expression


正则表达式执行的操作，可以粗略分为三大类：匹配（提取和验证）、替换、切分。

## 工具

* [RegExr is an online tool to learn, build, & test Regular Expressions (RegEx / RegExp).](https://regexr.com/)
* [Regester](https://deerchao.cn/tools/wegester/)
* [Python re 正则表达式操作](https://docs.python.org/zh-cn/3/library/re.html)
